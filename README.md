Задание второй недели курса Functional Programming Principles in Scala (Scala 2 version) с сайта coursera.org.

Необходимо реализовать набор функций, реализация которых помогает освоить основы функционального подхода.
Данное задание посвящено функциональным представлениям множеств.

На вход большинству функций в качестве одного из параметров передается функция (Int => Boolean), что позволяет работать с множествами.

Перед каждой функцией есть небольшое описание\требование, в файле FunSets.scala
